如何用hexo搭建个人网站

Posted on 2021-01-20

需要安装的东西

git, node.js, hexo, code

步骤

安装Git
安装Node.js
安装Hexo
GitHub创建个人仓库
生成SSH添加到GitHub
将hexo部署到GitHub
设置个人域名
发布文章

操作

安装好node.js，用win+r输入cmd进入命令行使用代码
node -v #查看node版本
npm -v #查看npm版本
然后安装淘宝的源
npm install -g cnpm –registry=http://registry.npm.taobao.org #安装淘宝的cnpm 管理器
cnpm -v #查看cnpm版本
cnpm install -g hexo-cli #使用cnpm安装hexo框架也可以用npm
hexo -v #查看hexo版本，验证是否安装成功
然后建立文件夹blog,我是直接在E盘新建文件夹，也可以用代码
mkdir blog #创建blog目录
建立好文件夹以后不要怕错，错了把文件夹删了重来就是了
然后在blog文件夹里右键选择git bash here,使用代码生成博客
hexo init #生成博客 初始化博客
安装完成以后blog目录会生成许多文件可以用代码查看
ls -l #查看目录下的文件
然后启动博客
hexo s #启动本地博客服务,hexo s指hexo server
启动以后会生成一个localhost:4000的链接，可以用浏览器访问
然后用Ctrl+C断开链接
http://localhost:4000/ #本地访问地址
hexo s一般用来启动博客预览
新建博客文章，也可以在blog\source_posts的目录右键新建markdown文件
hexo n “我的第一篇文章” #创建新的文章
给文件加内容可以用code（或者typora或者用vim)打开文件自己在里面自己写内容
然后回到blog目录
hexo clean #清理
hexo g #生成
hexo s #重新启动一下
然后刷新一下4000的那个网站，发现刚才添加的内容出现在网站上就成功了
测试成功以后用ctrl+c断开链接，进入github，建立一个仓库 用户名.github.io
然后在blog目录右键bash here输入代码
cnpm install –save hexo-deployer-git #在blog目录下安装git部署插件
使用code打开_config.yml修改配置
deploy:
type: git
repo: https://github.com/YourGithubName/YourGithubName.github.io.git
branch: master
然后部署到远端
hexo d #部署到Github仓库里
可能需要输入github账号和密码输入就行了
然后复制你的仓库名在浏览器访问能打开就成功了
https://YourGithubName.github.io/ #访问这个地址可以查看博客
可能还需要ssh公钥配置完就可以了，如何配置参考https://zhuanlan.zhihu.com/p/44213627
下载主题还是在blog目录可能比较慢
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia #下载yilia主题到本地
下载完成以后会在themes里有个yilia文件
继续使用code进入_config.yml修改使用的主题
把theme默认的landscape换成yilia
hexo clean #清理一下
hexo g #生成
hexo d #部署到远程Github仓库
打开https://YourGithubName.github.io/ 刷新几次主题就有了
自定义域名我是在阿里云购买的链接：https://wanwang.aliyun.com/domain/
购买完把你的域名丢到github里，设置界面里的custom domain里填写你的域名，勾选enforce HTTPS就能生成安全链接了，就可以用自定义的域名进入你的博客了。
